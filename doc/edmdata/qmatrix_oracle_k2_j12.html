qmatrix_oracleR Documentation

Oracle Q Matrices

Description

Pre-generated identified Q matrices used in simulation studies to verify recovery.

Format

A matrix with varying numbers of traits (K) and items (J).

Specifically:

  • qmatrix_oracle_k2_j12: 12 items and 2 traits.

  • qmatrix_oracle_k3_j20: 20 items and 3 traits.

  • qmatrix_oracle_k4_j20: 20 items and 4 traits.

  • qmatrix_oracle_k5_j30: 30 items and 5 traits.

Details

Each entry in the matrix is either 1, if the item uses the skill, or 0, if the item does not use the skill.
